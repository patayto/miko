The practice of using Python to programmatically generate, synthesize, and manipulate music. Encompasses three main approaches:

**Procedural Generation**: Using algorithms like [[Markov Chains]] to generate sequences based on probabilities, random walks, or rule systems. Similar to [[Yoga Flow Generation]] but applied to melodies, harmonies, and rhythmic patterns.

**MIDI Sequencing**: Manipulating MIDI data to compose or transform musical sequences. Libraries like `music21` provide abstractions for working with musical notation, pitch, duration, and timing.

**Audio Synthesis**: Real-time sound generation using waveform synthesis, oscillators, and effects. Python libraries like `pydub` and `librosa` enable audio manipulation at the signal level.

**Key Libraries**:
- `music21`: Music theory and notation abstractions
- `librosa`: Audio analysis and feature extraction
- `pydub`: Simple audio manipulation
- `pretty_midi`: MIDI file handling
- `synthesis`/`synth`: Real-time waveform generation

Applications span interactive systems (procedural soundtracks), algorithmic composition (art music), and training data generation for ML models. The Python ecosystem offers accessible entry points while supporting sophisticated audio processing workflows.
