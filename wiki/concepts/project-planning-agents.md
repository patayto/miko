A planned [[Claude Code]] capability: an agent grounded in a curated set of planning textbooks and frameworks, producing consistent planning output across different scales and contexts.

The core design goal is adaptive detail: a software PRD when that's what's needed; a simple chore checklist when that's enough. Ideally this decision is automatic, derived from the nature of the query, rather than requiring explicit instruction. The agent reads the task and picks the level.

The motivation is consistency. Without a shared planning methodology, quality varies unpredictably from session to session. An agent anchored in solid textbook knowledge provides a stable baseline — planning support that feels like working with someone who has read the right books and internalised them, not someone making it up each time.
