React is a JavaScript library for building user interfaces, developed and maintained by Meta. It is component-based: UIs are composed from small, reusable pieces that manage their own state and render declaratively based on data. React introduced the virtual DOM diffing model to make UI updates efficient without direct DOM manipulation.

React's core programming model centres on functional components and hooks (introduced in v16.8). Hooks such as `useState`, `useEffect`, and `useContext` replace the older class-based lifecycle methods. The unidirectional data flow — props passed down, events bubbled up — is a deliberate constraint that makes state changes predictable and traceable.

React is typically paired with a build toolchain (Vite, Create React App) and can be used with routing libraries (React Router), state managers (Redux, Zustand), and meta-frameworks (Next.js, Remix) that add server-side rendering and file-based routing.
