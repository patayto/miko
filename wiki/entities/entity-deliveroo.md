Deliveroo is a food delivery technology company that conducts technical interviews for engineering roles. The company's coding assessment includes a structured two-section evaluation: a 30-minute multiple-choice section testing knowledge of [[REST API Design Fundamentals]], systems design, and [[Testing Fundamentals]], followed by a 70-minute live coding section with two medium-difficulty algorithmic problems.

The assessment emphasizes practical REST API knowledge including HTTP methods, status codes, pagination, query parameters, and authentication. Systems design coverage includes stateless architecture, caching strategies, scalability principles, and REST principles. The coding section focuses on [[Interview Coding Patterns]] with difficulty comparable to HackerRank or LeetCode medium-level problems, with some problems including API-related task variations.
