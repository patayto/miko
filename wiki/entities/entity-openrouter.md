A unified API marketplace and routing service that provides access to multiple LLM providers including [[Qwen Code]], Claude, and others through a single API interface. OpenRouter abstracts provider differences and enables developers to route requests based on cost, latency, or feature requirements.

OpenRouter is commonly used with [[LiteLLM]] as the underlying provider for models like Qwen, allowing developers to seamlessly integrate multiple models into their applications without managing separate API keys and endpoints for each provider.
