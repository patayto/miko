SuperClaude is an open-source framework for extending [[Claude Code]] with a structured library of specialised agents, personas, and workflows. The author encountered it and noted it as a reference implementation that closely mirrors their own prior design attempts — same instincts, further developed.

Each agent has its own definition file (e.g. `system-architect.md`) giving it a distinct role and personality. The project includes a technical architecture guide and an examples cookbook. It is the most concrete published example of the "accumulate a library of specialised agents" pattern the author wants to build.

See also: [[Claude Code]], [[Sub-agents]]
