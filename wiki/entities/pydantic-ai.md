A Python library for building agentic systems with async-native support and tight integration with Pydantic for data validation. pydantic-ai is used at [[Kraken]] for LLM agent workloads and serves as the Python ecosystem equivalent to the JavaScript AI-SDK.

The library is designed for structured, type-safe agent development, with built-in support for tool use, state management, and dependency injection. pydantic-ai works well alongside FastAPI and SQLAlchemy in Python web applications that need agentic capabilities.
