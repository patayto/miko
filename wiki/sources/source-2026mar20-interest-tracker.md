A project concept for a unified home page to track and monitor personal interests across music and film. The tracker would allow the user to register favorite artists, actors, directors, and other media entities, then aggregate notifications and alerts from multiple sources.

For **music**, the tracker would enable:
- Artist registration (e.g., BTS, Ariana)
- Tour date monitoring via [[Setlist.fm]] integration
- Release date alerts for new music
- Social media monitoring (Instagram posts and announcements)
- Detail pages per artist with upcoming events and releases

For **film**, the tracker would support:
- Filtering by actors, directors, genres, and franchises
- Upcoming film discovery and notification
- Similar interest-based views as the music side

The core value is centralized awareness: rather than manually checking multiple sources (artist Instagram, tour databases, film sites), the user has a single dashboard showing all upcoming events of interest. This addresses the scattered nature of media discovery and the effort required to stay aware of activity from favorite creators across platforms.

See also: [[Artist Monitoring]] (foundational concept for the music tracker), [[Film Discovery]] (foundational concept for the film tracker).
